# Collector Modules

Planned modules:

- `collect_events.py`: fetch active and closed Polymarket events/markets/tags.
- `collect_prices.py`: fetch CLOB order-book and latest price snapshots.
- `collect_trades.py`: fetch public trade data.
- `compute_whales.py`: compute relative large-trade signals per option.
- `export_r2.py`: export chart-ready JSON files to Cloudflare R2.
